# PRODIGY_WD_03

Tic-Tac-Toe Web Application

## Description
This project is a Tic-Tac-Toe web application developed as part of Task-03 of the internship.
It allows two players to play the game, detects winning conditions, and handles draw scenarios.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Two-player gameplay
- Win and draw detection
- Interactive UI
- Reset game functionality
